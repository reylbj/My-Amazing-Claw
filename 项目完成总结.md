## ✅ 2026-03-07 网关稳定性修复（`Unknown system error -11, read`）

### 问题现象
- 聊天框发送消息时报错：`Error: Unknown system error -11: Unknown system error -11, read`。

### 根因与处理
- 运行环境使用 `Node 25` 时，历史日志持续出现 `-11`。
- 已切换并固化到 `Node 22 LTS`，重装并重启网关服务，不改业务代码。

### 固化命令（已验证）
```bash
npm install -g node@22
openclaw gateway install --force
openclaw gateway restart
openclaw gateway status
```

### 验证标准
- `openclaw update status`：当前 OpenClaw 为 `2026.3.2`（latest）。
- `openclaw gateway status`：`RPC probe: ok`。
- `openclaw agent --agent main --message "ping" --json`：可正常返回。
- Dashboard 可访问：`http://127.0.0.1:18789/`。

### 后续统一启动方式（避免再犯错）
1. 只用 `openclaw gateway restart` + `openclaw gateway status` 启动与验活。
2. 发现 `-11` 先重启；仍异常再执行 `openclaw gateway install --force` 后重启。
3. 禁止手工 kill 网关或 `stop+start` 快速连击。
