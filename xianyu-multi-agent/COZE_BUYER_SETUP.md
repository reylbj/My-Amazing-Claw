## 📋 扣子智能体设置

### 1. 创建扣子智能体

1. 访问 [扣子平台](https://www.coze.cn/)
2. 创建新的智能体，设置如下：

**智能体名称**: 闲鱼买家助手  
**智能体描述**: 专业的闲鱼买家，擅长询问商品详情、议价谈判和购买决策

**人格设定**:
```
你是一个真实的闲鱼买家，有以下特点：

身份：普通消费者，对商品感兴趣但很谨慎
性格：理性、节俭、会货比三家
说话风格：自然随意、口语化、简洁明了
购买习惯：会询问详情、适度议价、理性决策

行为规则：
1. 说话要像真人，别太正式
2. 每次只问1个关键问题，不要贪多  
3. 会用"嗯"、"啊"、"哦"等语气词
4. 回复控制在15-25字
5. 不要用引号包围回复
6. 展现真诚的购买意向

你的回复应该自然、简洁、有人情味。
```

**技能配置**:
- 商品询问：询问成色、功能、配件等
- 价格谈判：合理议价，不过分压价
- 购买决策：基于预算和需求做决定

### 2. 环境变量配置

在你的 `.env` 文件中添加扣子相关配置：

```bash
# 扣子API配置（买家AI专用）
COZE_API_KEY=your_coze_api_key_here          # 扣子API密钥
COZE_BASE_URL=https://api.coze.cn            # 扣子API地址
COZE_BUYER_BOT_ID=your_buyer_bot_id_here     # 买家智能体ID

# 原有配置保持不变（卖家AI继续使用）
API_KEY=your_openai_api_key                   # 卖家AI使用
MODEL_BASE_URL=https://dashscope.aliyuncs.com/compatible-mode/v1
MODEL_NAME=qwen-max

# 买家AI系统配置
BUYER_MAX_BUDGET=5000                         # 买家最大预算(元)
BUYER_TARGET_DISCOUNT=0.8                     # 目标折扣比例
BUYER_MAX_ROUNDS=3                            # 最大议价轮次
```

### 3. 获取扣子API密钥

1. 在扣子平台右上角点击头像
2. 进入「API管理」
3. 创建新的API密钥
4. 复制密钥到环境变量

### 4. 获取智能体ID

1. 在你创建的智能体页面
2. 点击「发布」→「API访问」
3. 复制Bot ID到环境变量

## 🔄 系统架构

```
买家消息 → 扣子智能体 → 自然回复
卖家消息 → 通义千问    → 专业回复
```

- **买家AI**: 使用扣子智能体，更自然的对话体验
- **卖家AI**: 继续使用通义千问，保持现有功能

## 🚀 启用说明

1. 配置完成后，系统会自动检测扣子配置
2. 如果扣子配置正确，买家AI将使用扣子智能体
3. 如果扣子配置缺失，会自动fallback到通义千问
4. 卖家AI功能完全不受影响

## 🐛 故障排除

**问题1**: 扣子API调用失败
- 检查API密钥是否正确
- 确认智能体ID是否有效
- 检查网络连接

**问题2**: 回复质量不理想
- 优化智能体的人格设定
- 调整技能配置
- 丰富训练示例

**问题3**: 响应速度慢
- 检查扣子API服务状态
- 考虑调整timeout设置

## 🔄 会话管理机制

扣子智能体的关键优势是能维持对话连续性。系统通过以下方式管理会话：

### 会话ID生成规则

```bash
conversation_id = f"buyer_{item_id}_{hash(chat_id) % 1000}"
```

**确保同一商品的所有对话都在同一个扣子会话中进行**

### 会话分离逻辑

| 维度 | 说明 | 示例 |
|------|------|------|
| 商品级别 | 不同商品 = 不同扣子会话 | 商品A和商品B分开 |
| 聊天级别 | 不同买家 = 不同扣子会话 | 张三和李四分开 |
| 角色级别 | 买家用扣子，卖家用通义千问 | 双引擎架构 |

### 实际效果

**✅ 正确的会话连续性:**
```
扣子会话ID: buyer_12345_678
第1轮: 买家: "嗯，东西成色怎么样？"
第2轮: 卖家: "九成新，功能都正常"  
第3轮: 买家: "那包邮吗？" (记住了之前的对话)
```

**❌ 错误的会话分离:**
```
扣子会话ID: buyer_hash1
第1轮: 买家: "嗯，东西成色怎么样？"

扣子会话ID: buyer_hash2  
第2轮: 买家: "东西成色怎么样？" (忘记了之前问过)
```
