# 贡献指南 | Contributing Guide

感谢你对 AI 投资 Agent 的关注！欢迎提交 Issue 和 PR。

## 如何贡献

### 报告 Bug

1. 先搜索 [已有 Issue](https://github.com/AIPMAndy/ai-invest-agent/issues) 避免重复
2. 使用 Bug Report 模板创建新 Issue
3. 提供复现步骤、期望行为、实际行为

### 提交功能建议

1. 使用 Feature Request 模板创建 Issue
2. 说明使用场景和预期效果

### 提交代码

1. Fork 本仓库
2. 创建特性分支：`git checkout -b feature/your-feature`
3. 提交修改：`git commit -m 'Add: your feature'`
4. 推送分支：`git push origin feature/your-feature`
5. 创建 Pull Request

### Commit 规范

```
Add: 新增功能
Fix: 修复问题
Update: 更新内容
Remove: 删除内容
Docs: 文档修改
```

## 开发环境

```bash
git clone https://github.com/AIPMAndy/ai-invest-agent.git
cd ai-invest-agent
pip install -r requirements.txt
```

## 代码规范

- Python 代码遵循 PEP 8
- Markdown 文件使用 UTF-8 编码
- 配置文件使用 JSON 格式

## 许可

提交 PR 即表示你同意将贡献内容按项目许可证发布。
