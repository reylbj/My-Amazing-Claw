# OpenClaw 项目总结

## 项目概述
OpenClaw 是为 Gemini API 免费层级设计的智能管理系统，核心理念：**解放双手，但绝不越过安全底线**。

---

## 核心功能

### 1. 三级安全体系
- **🔴 禁止**：数据删除、未授权通信、权限修改、财务操作
- **🟡 确认**：大规模修改、发送消息、系统配置
- **🟢 允许**：只读操作、临时文件、本地通知

### 2. 智能路由
- 自动任务复杂度分析（0-10分）
- 多因素加权评估
- 配额感知的智能降级

### 3. 配额管理
- 实时监控（RPM/RPD/TPM）
- 四级预警系统
- 时间分段配额分配
- 应急储备机制

### 4. 性能优化
- 批处理：节省40-60%
- 缓存：减少20-40%
- 本地处理：节省50-70%
- 综合效果：节省60-80%

---

## 系统架构

```
用户请求
  ↓
安全检查 (HEARTBEAT.md)
  ↓
智能路由 (smart_router.py)
  ↓
配额检查 (quota_monitor.py)
  ↓
批处理优化 (可选)
  ↓
缓存检查 (可选)
  ↓
API调用 (Gemini)
  ↓
记录审计
```
